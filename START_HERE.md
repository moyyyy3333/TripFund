# 🚀 START HERE - TripFund Quick Launch

**Welcome to your new TripFund app!** Here's how to get it running in the next 5 minutes.

---

## ⚡ Fastest Path to Running App

### 1. Install Dependencies (2 minutes)

```bash
cd TripFund/mobile
npm install
```

### 2. Start Development Server (1 minute)

```bash
npm start
```

This opens Expo Dev Tools in your browser.

### 3. View on Your Phone (2 minutes)

**iPhone or Android:**
1. Install **Expo Go** app from App Store / Play Store
2. Scan the QR code from Expo Dev Tools
3. App loads on your phone!

**OR use simulator:**
```bash
npm run ios      # Mac only
npm run android  # Any OS with Android Studio
```
