# TripFund Web App

**Premium web version of TripFund - accessible from any browser, any device.**

---

## ✨ Features

- 🎨 Premium dark theme with vibrant gradients
- 📱 Fully responsive (desktop, tablet, mobile)
- ⚡ Lightning-fast Next.js 14 + App Router
- 🔐 Supabase authentication ready
- 🎯 Matches mobile app design exactly
- 🚀 Deployed on Vercel

---

## 🚀 Quick Start

### Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

### Production Build

```bash
npm run build
npm start
```

## 📦 Tech Stack

- **Framework:** Next.js 14
- **Language:** TypeScript

- **Fonts:** Geist Sans & Geist Mono

## 🎨 Design System

Matches the mobile app exactly:
- Dark mode by default
- Teal (#00d9ff) and Coral (#ff6b6b) accents
- Smooth animations and transitions
- Travel-focused gradients

---

## 🔧 Environment Setup

Create `.env.local`:

```env
NEXT_PUBLIC_SUPABASE_URL=your-supabase-url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your-anon-key
```
